Qui suis-je?

Sur moi

Hello, moi c'est Jordy ELLANGMANE, ans, divorcé. Voici mon parcours.

Quand j'ai obtenu mon diplôme d'ingénieur des travaux informatique, le marché du travail était au plus bas, il était quasiment impossible de trouver un job sans expérience professionnelle. Bref, c'était la crise ! Très peu d'entreprise confient leurs projets aux jeunes Développeurs sortant à peine des écoles sans aucune expérience professionnelle ou réalisations.

Je vous rassure, je ne me suis pas laissé abattre.
J'étais jeune et j'avais des rêves plein la tête. Cette situation m'a donné un énorme coup de boost, je devais d'abord créer mon portfolio.
Le petit souci, c'est que les études universitaires ne m'avaient pas du tout préparé à la réalité. J'avais acquis surtout des connaissances théoriques : beaucoup de Maths, beaucoup de Physique, beaucoup de théorie sur le monde informatique.
Mais rien de tout ça n'allait m'aider à construire mon produit et c'est pourquoi j'ai commencé à me former online.
Mais au final, ce « metro-boulot-dodo » ne me convenait plus. J'avais besoin de nouveau et surtout j'avais besoin de plus de liberté. J'ai donc décidé de me lancer en freelance et j'ai créé Dix sur Dix.

Quest-ce que je sais faire?

Mes outils
javaScript

JavaScript est un langage de programmation de scripts principalement employé dans les pages web interactives et à ce titre est une partie essentielle des applications web. Avec les langages HTML et CSS, JavaScript est au cœur des langages utilisés par les développeurs web.

React

Est une bibliothèque JavaScript libre développée par Facebook depuis 2013. Le but principal de cette bibliothèque est de faciliter la création d'application web monopage, via la création de composants dépendant d'un état et générant une page (ou portion) HTML à chaque changement d'état.

nodeJs

Node.js est une plateforme logicielle libre en JavaScript, orientée vers les applications réseau utilisant le moteur javascript de chrome v8.
Node.js est utilisé notamment comme plateforme de serveur Web

Bootstrap 5

Bootstrap est une collection d'outils utiles à la création du design (graphisme, animation et interactions avec la page dans le navigateur, etc.) de sites et d'applications web. C'est un ensemble qui contient des codes HTML et CSS.

php

Est un langage de programmation libre, principalement utilisé pour produire des pages Web dynamiques via un serveur HTTP.

Laravel

Laravel est un framework web open-source écrit en php respectant le principe modèle-vue-contrôleur et entièrement développé en programmation orientée objet.

SQL

Est un langage informatique normalisé servant à exploiter des bases de données relationnelles. La partie langage de manipulation des données de SQL permet de manipuler des données dans les bases de données relationnelles..

firebase

Firebase est un ensemble de services d'hébergement pour n'importe quel type d'application (Android, iOS, Javascript, Node.js, Java, Unity, PHP, C++ ...). Il propose d'héberger en NoSQL et en temps réel des bases de données.

git

Git est un logiciel de gestion de versions décentralisé.
